written from zero

No forks, no templates, every one of these started as an empty directory. Operating systems to understand machines, a language to understand meaning, engines to understand speed, and a security toolkit because defending a system is the final exam for understanding it. One page per project below.

all repos on github →

01 · makaos

cx86-64 desktop · WIP

A 64-bit OS for x86-64, from the UEFI handoff up: a 4-way SMP kernel with demand-paged memory, AHCI/NVMe storage, a TCP/IP stack, ext2 read/write, and a ring-3 userland speaking ~118 Linux-style syscalls that runs bash and DOOM. A ported sway/wlroots desktop is landing now and still being stabilized.

02 · veil

golinux open core · early access

A per-application network isolation engine. Confines any process to its own Linux network namespace and routes it through a composable WireGuard, OpenVPN, Tor, SOCKS5 and HTTP chain, behind a verified, fail-closed kill switch. Open-core: Veil Core is free and open source; Veil Pro adds the anti-detect and operational-security stack.

03 · voxelg

rustwgsl gpu raymarched

A GPU-raymarched voxel engine: hierarchical DDA in a compute shader, Gerstner water, dispersive glass, volumetric clouds, cellular-automaton physics, and lock-free TCP multiplayer with seed-synced worlds.

04 · maka

rustc v1 · tested

A systems language that puts ownership and nullability in the type: five pointer kinds, forced null handling with no runtime checks, structural attr/has traits, pattern-matched enums, lightweight concurrency, and inline C and Rust interop. Compiles to portable C; compiler in Rust, standard library in Maka.

pwsh/bash

A menu-driven defensive-security toolkit for Windows and Linux. One workflow: install, map the host, get a prioritized punch list, harden (dry-run first), triage, hunt, then roll it all into a graded A-F security report card. Authorized use only.

python graded a-f

A security report card for any domain: TLS, HTTP security headers, and SPF/DMARC/CAA, graded a to f with a concrete fix for every finding. Queries a public resolver so split-horizon DNS can't hide records. Stdlib only.
