01 · makaos

cnasm 146/146 tests

A 64-bit operating system for x86-64, from the boot sector up: two-stage NASM bootloader, C kernel with preemptive scheduling and demand-paged memory, AHCI storage, ext2 read/write, and ring-3 userspace speaking 29 Linux-style syscalls. A Wayland desktop is in progress.
